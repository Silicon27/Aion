Thank you for your interest in contributing to our project! We welcome contributions from the community. Please follow these guidelines to ensure a smooth process.\hypertarget{md_core_2include_2support_2si_2_c_o_n_t_r_i_b_u_t_i_n_g_autotoc_md4}{}\doxysection{\texorpdfstring{Layout of Si}{Layout of Si}}\label{md_core_2include_2support_2si_2_c_o_n_t_r_i_b_u_t_i_n_g_autotoc_md4}
All code for Si is located in the \mbox{[}{\ttfamily include/}\mbox{]}(include) directory. There are no source files, only headers. Each header file should contain a single implementation of a data structure or algorithm.

Si headers are organized as a header file with includes, which can be found in the \href{include/si}{\texttt{{\ttfamily include/si/}}} directory. Each Si header should have its own header file, and the file name should match the Si header name.
